## Current state of the tool

As of HEAD (801b190, 2026-07-07, synced to origin/main): the tool has evolved from a review-analysis dashboard into **"The Factory" — a multi-agent creative-brief production system** wrapped in a warm-premium (cream/navy, Newsreader/Inter) shell with persistent sidebar. The core pipeline (src/autopilot/pipelineEngine.ts) runs Strategy Session → Creative Strategist (inspiration-mode-aware, receives product truth + Visual Craft Guide + Claim Boundary) → Concept Generation → Differentiation Critic (with hard Claim Grounding gate) → Concept Evaluator/Selector → Script Writer → Batch Reviewer (11-criterion rubric + brand-fact hard gate) → Memory Curator, with two batch-entry paths (Asana screenshot vision-parse or manual table). Three additive knowledge layers now feed every creative call: (1) embedded manifesto/angle/objection banks in src/prompts/, (2) the Brain (src/brain/ — VoC index from reviews + Meta comments, per-module flags, session-cached deep reasoning, window.__brain.killAll() kill switch), (3) the Marketing Brain (src/knowledge/brain/ — 39.5k-word studied-source distillations routed per-step via src/prompts/marketingBrain.ts, ~$0.5-0.6/brief cost). Models are centralized in src/config/models.ts as three tiers: IDEATION='claude-opus-4-8' (intended 'claude-fable-5', reverted Jun 16 for key-access reasons — flip back is one line), CREATIVE='claude-opus-4-8', UTILITY='claude-sonnet-4-6'. Brief output uses 4 templates (ecom/agc/ugc/fullai) with strict 1:1 template→preview→download mapping and right-anchored column-drift-resistant parsing. Supporting modules: Ad Comment Intelligence with live Meta Graph pulls through a read-only (GET-only-enforced) Cloudflare Worker meta-proxy with 4-strategy page-token discovery, batched 10K-comment analysis, and a saved/combinable analyses library; Inspiration Bank (folder tabs, short-form punch-through, two-tier 10-reference selection); Memory Vault; Customer Reviews dashboard; Context Hub; plus the original Segments/Personas/Product Intelligence insights modules. Storage is entirely browser-side IndexedDB (multiple DBs — note the cross-file DB_VERSION invariant between vocIndex.ts and commentAnalysisStore.ts that already caused one regression). The newest commits show the current battle line: prompt-engineering governance — grounding agents in studied sources and preventing claim invention for data-thin products.

## Repo conventions

**Branching/release:** single `main` branch, zero tags, no PRs or merge commits — every change committed directly to main and pushed; origin/main tracks HEAD exactly. Deploy is push-triggered GitHub Pages via .github/workflows/deploy.yml; the two Cloudflare Workers (proxy/, meta-proxy/) deploy separately via wrangler and are versioned in-repo. **Commit messages:** imperative subjects, frequently module-prefixed ("Briefs:", "Inspiration:", "Sidebar:", "Full AI:") or em-dash-subtitled, followed by exceptionally detailed multi-paragraph bodies — root-cause analysis, file-by-file change lists, explicit "what stays untouched" scoping, rollback instructions, and expected-impact estimates.
